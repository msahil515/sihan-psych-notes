# Ch 3: Memory: Systems, Working Memory & Forgetting

**Overview:** Memory involves three stages: **encoding** (processing during presentation), **storage** (retaining information), and **retrieval** (recovering stored information). Forgetting = failed retrieval. Covers architecture (multi-store to working memory), depth of processing, forgetting mechanisms, long-term memory systems (declarative vs. non-declarative), and everyday memory (autobiographical memory, eyewitness testimony, prospective memory). Source: Chapters 6, 7, 8.

---

## 3.1 Architecture of Memory: The Multi-Store Model

**Atkinson and Shiffrin (1968)** proposed three distinct stores:

| Store | Capacity | Duration | Code |
|-------|----------|----------|------|
| **Sensory stores** | Large but modality-specific | ~500 ms (iconic); ~2 s (echoic) | Modality-specific |
| **Short-term store (STS)** | ~7 items (Miller, 1956); true capacity ~3-4 chunks | Seconds without rehearsal | Mainly phonological |
| **Long-term store (LTS)** | Essentially unlimited | Minutes to decades | Semantic + other |

- **Iconic memory** (visual sensory store): decays within ~500 ms; Sperling (1960). Recent evidence (Persuh et al., 2012) shows it is NOT fully preattentive; attentional load disrupts storage.
- **Echoic memory** (auditory sensory store): ~2 s; longer in the left hemisphere (up to 5 s) than right (2 s), reflecting left-hemisphere dominance for language (Ioannides et al., 2003).
- **Chunks**: integrated units of information. True STM capacity = 3-4 chunks without rehearsal (Chen & Cowan, 2009). Word span can reach 20 if words form sentences (Simon, 1974) because of chunking via LTM.
- **Serial position effects**: Items at the start of a list are better recalled (**primacy effect**) because they have been rehearsed into LTM. Items at the end (**recency effect**) remain in the short-term store. The dissociation supports the two-store distinction.
- **Rehearsal** in STS was assumed to transfer information to LTM; multi-store greatly overstated its role.

### Key Evidence: Double Dissociation

| Patient | STS | LTM | Implication |
|---------|-----|-----|-------------|
| **HM** (Henry Molaison; Scoville & Milner, 1957) | Intact digit span | Severe anterograde amnesia; no new declarative LTM | LTM distinct from STS |
| **KF** (Shallice & Warrington, 1970) | Digit span of ~2 (severely impaired STS) | Intact long-term learning and recall | STS distinct from LTM |

> **Key Study: KF (Shallice & Warrington, 1970)** - Patient KF had no problems with long-term learning and recall but had a very small digit span. His STS deficit was mainly verbal (letters, words, digits) rather than meaningful sounds or visual stimuli. This double dissociation with amnesics (intact STS, impaired LTM) provides strong support for separate stores.

### Criticisms of the Multi-Store Model
1. Both stores are assumed to be **unitary** (single, uniform) - incorrect; replaced by working memory system (STS) and multiple LTM systems.
2. STS is not a pure gateway; information from LTM is already active when we use STS (e.g., chunking "IBM" requires LTM access).
3. Assumption that only consciously processed material reaches LTM is disproved by **implicit learning**.
4. Not all items in STS have equal status; the currently attended item is accessed faster (McElree, 2006).
5. Rehearsal is only one of many routes to LTM; the vast majority of long-term memories were never deliberately rehearsed.

## 3.2 Working Memory (Baddeley & Hitch, 1974)

**Baddeley and Hitch (1974)** replaced the unitary STS with a **working memory** system. The current model (Baddeley, 2012) has **four components**:

```
Central Executive (attentional controller)
        |              |              |
Phonological    Visuo-spatial    Episodic
   Loop          Sketchpad        Buffer
        \              |              /
         ----- Long-Term Memory -----
```

### 3.2a Phonological Loop

Two subcomponents:
- **Phonological store** (passive): holds speech-based traces; directly linked to speech perception.
- **Articulatory rehearsal process** (articulatory control process): subvocal rehearsal, linked to speech production; refreshes the phonological store.

Key phenomena supporting the phonological loop:

| Effect | Description | Implication |
|--------|-------------|-------------|
| **Phonological similarity effect** | Immediate serial recall is ~25% worse for phonologically similar words (FEE, HE, KNEE) vs. dissimilar (BAY, HOE, IT) (Larsen et al., 2000) | Items are held in a phonological code; similar sounds cause confusion |
| **Word-length effect** | Memory span is greater for short words than long words (Baddeley et al., 1975) | Longer words take longer to rehearse; fewer fit in the rehearsal loop |
| **Articulatory suppression** | Repeating "the the the" eliminates the word-length effect for visually presented words | Confirms word-length effect depends on rehearsal; disrupts the articulatory process |

- **Function of the phonological loop**: Learning new vocabulary (especially foreign words). Patient PV (digit span of 2) showed no learning of Russian-Italian word pairs but normal learning of Italian-Italian pairs (Baddeley et al., 1998). Articulatory suppression impairs foreign vocabulary learning (Papagno et al., 1991).

### 3.2b Visuo-Spatial Sketchpad (VSSP)

- Stores and manipulates **visual patterns and spatial movement**.
- Logie (1995) proposed two subcomponents:
  - **Visual cache**: stores visual form and colour.
  - **Inner scribe**: processes spatial and movement information; rehearses the visual cache; transfers to central executive.
- Smith and Jonides (1997): separate brain activation for spatial tasks (right hemisphere) vs. visual tasks (left hemisphere), supporting the dual-component view.
- Klauer and Zhao (2004): spatial interference (movement task) disrupts spatial main task more than visual main task; visual interference (colour task) disrupts visual main task more. Supports separable visual and spatial sub-systems under low attentional load.

### 3.2c Central Executive

- **Modality-free**, limited-capacity attentional system; the most important and versatile component.
- Does NOT store information; coordinates the slave systems and performs complex cognitive tasks.
- Brain region: **dorsolateral prefrontal cortex** (BA9/46) most strongly associated; but executive processes also involve anterior cingulate cortex and other areas (Stuss, 2011).

**Miyake et al. (2000) - Three Executive Functions:**

| Function | Description | Example Task |
|----------|-------------|--------------|
| **Inhibition** | Override dominant responses; resist distraction | **Stroop task** (name ink colour when word says different colour) |
| **Shifting** | Switch flexibly between tasks or mental sets | Alternating multiplication and division |
| **Updating** | Monitor and rapidly add/delete working memory contents | Keep track of most recent items across categories |

- Miyake and Friedman (2012) **unity/diversity framework**: each executive function = common component + unique component. Inhibition has almost no unique variance (it is nearly entirely the common component). Both genetic and neuroimaging evidence support the framework (Collette et al., 2005; Friedman et al., 2008).
- **Dysexecutive syndrome** (Baddeley, 1996): frontal lobe damage causing impaired inhibition, rule deduction, set maintenance and shifting. Patients have great difficulty holding jobs and functioning in daily life.

### 3.2d Episodic Buffer

- Added by Baddeley (2000); holds **integrated, multi-dimensional information** (combining visual, auditory and other codes) as **chunks** or episodes.
- Capacity: approximately **4 chunks**.
- Acts as a buffer between working memory components and LTM; enables immediate sentence recall (up to ~16 words) far beyond the phonological loop's capacity.
- Currently understood to be an **essentially passive structure** on which bindings achieved elsewhere are displayed (Baddeley, 2012). Integration of features does not require the central executive (Allen et al., 2012).

### 3.2e Working Memory Capacity (Individual Differences)

- **Reading span** (Daneman & Carpenter, 1980): largest number of sentences read for comprehension from which all final words can be recalled; measures simultaneous processing + storage.
- **Operation span**: maximum items for which participants can recall all final words while verifying arithmetic problems.
- Working memory capacity correlates **+0.6 with intelligence**, especially **fluid intelligence** (Unsworth, 2010). Low correlation with crystallised intelligence (knowledge-based).
- Engle and Kane (2004) **two-factor theory**: high-capacity individuals excel at (1) maintaining task goals and (2) resolving response conflict/competition.
- High-capacity individuals have greater **attentional control**: less disruption by external distraction (Sorqvist, 2010) and less mind-wandering (McVay & Kane, 2012).
- Anxiety reduces working memory capacity (Eysenck et al., 2007); relaxation training enhances it (Kargar et al., 2013).

## 3.3 Levels of Processing (Craik & Lockhart, 1972)

**Core claim**: The **depth or level of processing** at encoding - not simply the duration or amount of rehearsal - determines long-term memory strength.

| Level | Type of Analysis | Example Task |
|-------|-----------------|--------------|
| **Shallow (structural/graphemic)** | Physical features | Is the word in uppercase? |
| **Intermediate (phonemic)** | Sound | Does this word rhyme with "train"? |
| **Deep (semantic)** | Meaning | Does this word fit the sentence "She walked past the ___"? |

> **Key Study: Craik and Tulving (1975)** - Used incidental learning (no prior warning of memory test). Deep (semantic) processing produced more than **three times** higher recognition memory than shallow (graphemic) processing. Concluded that memory depends on the nature of processing, not intention to learn.

### Elaboration and Distinctiveness

- **Elaboration of processing**: More elaborate semantic processing = better memory. Complex sentence frames (e.g., "The great bird swooped down and carried off the struggling ____") produced twice the cued recall of simple frames (e.g., "She cooked the ____") (Craik & Tulving, 1975).
- **Distinctiveness**: A memory trace differs from others because it was processed differently. Eysenck and Eysenck (1980): pronouncing irregular words distinctively (e.g., the "b" in "comb") produced recall as good as deep semantic processing, even though the task was shallow. Distinctiveness can compensate for shallow depth.
- **Transfer-appropriate processing** (Morris et al., 1977): Memory depends on the match between processing at encoding and processing required at test. On a rhyming recognition test, shallow (rhyme) processing outperformed deep (semantic) processing - contradicting strict levels-of-processing theory.

### Criticisms
1. Underestimates the role of **retrieval environment** (Morris et al., 1977).
2. "Depth," "elaboration," and "distinctiveness" lack precise operational definitions.
3. Does not clearly explain WHY deep processing improves memory.
4. Levels-of-processing effects are **smaller for implicit memory** (e.g., word-fragment completion) than for explicit memory.

## 3.4 Learning Through Retrieval (Testing Effect)

**Testing effect**: Devoting part of the learning period to retrieving (testing) information enhances long-term memory more than additional study alone.

> **Key Study: Roediger and Karpicke (2006)** - Participants read a prose passage then either: (1) restudied four times, (2) read three times then recalled once, or (3) read once then recalled three times. After **5 minutes**, repeated study was best. After **1 week**, average recall in the repeated-test condition was **50% higher** than in the repeated-study condition - a dramatic reversal.

- **Mediator effectiveness hypothesis** (Pyc & Rawson, 2010): Testing forces learners to generate better mediators (e.g., link "wingu" to "plane" to recall "cloud"). Retrieval practice helps learners evaluate and replace ineffective mediators.
- **Retrieval effort hypothesis** (Pyc & Rawson, 2009): Testing is more effective when retrieval is difficult (spaced trials are better than massed).
- Dunlosky et al. (2013) reviewed ten learning techniques; repeated testing was the **most effective** of all.
- Brain regions for language (left inferior frontal gyrus) are more active during testing than restudying (van den Broek et al., 2013).

## 3.5 Implicit Learning

**Implicit learning**: Acquisition of complex information without conscious awareness of what has been learned (Cleeremans & Jiménez, 2002).

- Reber (1993) proposed implicit learning is: (1) age-independent, (2) IQ-independent, (3) robust to disorders (e.g., amnesia), (4) shows low individual variability, (5) common across species.
- Evidence for IQ independence: Intelligence correlates +0.44 with explicit learning but only +0.16 with implicit learning (Kaufman et al., 2010).
- **Serial reaction time task**: Participants respond to stimuli in a covert repeating sequence and become faster without awareness. Slowdown when a novel sequence appears reveals learning. However, many participants have partial conscious awareness (Wilkinson & Shanks, 2004; Gaillard et al., 2009).
- **Process-dissociation procedure**: Inclusion condition (guess next stimulus) vs. exclusion condition (try NOT to guess accurately). Difference = amount of explicit knowledge.

### Neural Basis
- **Striatum** (basal ganglia): associated with implicit learning; Parkinson's disease (striatal damage) impairs implicit learning, especially with error-correcting feedback (Foerde & Shohamy, 2011).
- **Dorsolateral prefrontal cortex + anterior cingulate**: associated with explicit/conscious learning (Dehaene & Naccache, 2001).
- Real-world example: **Skilled typists** (Snyder et al., 2014) could only locate 57% of keyboard letters on a blank keyboard, yet type ~70 words/minute - illustrating vast implicit learning underlying expert performance.

## 3.6 Forgetting From Long-Term Memory

### Ebbinghaus Forgetting Curve
- Ebbinghaus (1885/1913) studied forgetting using the **savings method** (reduction in relearning trials vs. original learning), using himself as subject and nonsense syllables as material.
- Pattern: Forgetting is **very rapid** over the first hour, then slows substantially. Rubin and Wenzel (1996): best described by a **logarithmic function**; Averell and Heathcote (2011) favour a **power function**.
- Implicit memory shows slower forgetting than explicit memory (Tulving et al., 1982); Mitchell (2006) found implicit memory for pictures from 17 years earlier.

### Major Theories of Forgetting

| Theory | Mechanism | Key Evidence |
|--------|-----------|--------------|
| **Decay** | Gradual loss of memory trace over time; mostly at short retention intervals; hippocampus most active site (Hardt et al., 2013) | Campoy (2012): strong decay at <3.3 s; less at longer intervals |
| **Proactive interference (PI)** | Prior learning disrupts memory for new learning | Learning A-B then A-C impairs recall of A-C; strong incorrect responses resist exclusion (Jacoby et al., 2001) |
| **Retroactive interference (RI)** | New learning disrupts memory for prior learning | Bilinguals name pictures slower in native language after naming in second language (Misra et al., 2012) |
| **Cue-dependent forgetting** | Lack of appropriate retrieval cues; inadequate encoding-retrieval overlap | Godden & Baddeley (1975): deep-sea divers recalled 40 words better on land (where learned) vs. underwater |
| **Consolidation failure** | Newly formed memories are physiologically fragile; disruption before consolidation completes causes forgetting | Alcohol before learning impairs; alcohol shortly after learning can IMPROVE memory (protects from interference) (Moulton et al., 2005) |
| **Repression / motivated forgetting** | Anxiety-provoking memories kept from conscious awareness (Freud); think/no-think paradigm: active suppression via inhibitory control (Anderson & Green, 2001) | Suppression linked to increased dorsolateral PFC + decreased hippocampal activation (Depue et al., 2010) |

> **Key Study: Godden and Baddeley (1975)** - Deep-sea divers learned 40 words on a beach or 10 feet underwater, then recalled in the same or opposite environment. Recall was substantially better when learning and test environments matched - powerful evidence for **cue-dependent (context-dependent) forgetting** and Tulving's **encoding specificity principle**.

### Encoding Specificity Principle (Tulving, 1979)
- "The probability of successful retrieval is a monotonically increasing function of informational overlap between information at retrieval and information stored in memory."
- Context can be **external** (physical environment) or **internal** (mood state = mood-state-dependent memory).
- Dramatic prediction: **recall can sometimes exceed recognition** when recall cues overlap more with the memory trace. Muter (1978): participants recognised 29% of famous names but recalled 42% when given first name + description.
- Context effects are generally larger for **recall** than for **recognition** (Godden & Baddeley, 1975 vs. 1980).

### Proactive vs. Retroactive Interference (Details)

| Type | Direction | Example |
|------|-----------|---------|
| **Proactive (PI)** | Old learning → impairs new | Munsterberg's watch pocket: old habit (response A) intrudes on new habit (response B) |
| **Retroactive (RI)** | New learning → impairs old | Exposure to foreign language reduces recall of native-language words |

- PI is reduced by instructing participants to **forget** prior lists - this restricts retrieval search to relevant items (Bäuml & Kliegl, 2013).
- Both PI and RI are maximal when different responses are paired with the same stimulus.
- RI can occur even when no new learning occurs - any **mental effort** during the retention interval causes some RI (Dewar et al., 2007).

### Consolidation and Reconsolidation

- **Consolidation**: Long-lasting physiological process; new memories are fragile for hours. Hippocampus is key early site; over time, memories migrate to the neocortex (McKenzie & Eichenbaum, 2011).
- Predicts a **temporal gradient in retrograde amnesia**: most forgetting for events just before amnesia onset (Manns et al., 2003; Squire & Wixted, 2011).
- **Reconsolidation**: Reactivating a consolidated memory makes it fragile again, allowing it to be updated or altered (Hardt et al., 2010). Hupbach et al. (2007, 2008): reactivating Time 1 memories during Time 2 learning causes Time 2 material to intrude into Time 1 memory. Explains misinformation effect in eyewitness testimony (Chan & LaPaglia, 2013).
- Sleep consolidation: cued reactivation of memories during sleep (auditory or olfactory cues) promotes consolidation and improves subsequent memory (Oudiette & Paller, 2013).

## 3.7 Long-Term Memory Systems

### Declarative vs. Non-Declarative Memory

```
Long-Term Memory
├── Declarative (Explicit) - conscious recollection
│   ├── Episodic memory - personal events, "when and where"
│   └── Semantic memory - general world knowledge
└── Non-Declarative (Implicit) - behaviour without conscious recollection
    ├── Procedural memory - knowing how (skills)
    ├── Priming - facilitated processing via prior exposure
    └── Simple classical conditioning / habituation
```

**Key distinction**: Declarative memory = **knowing that**; procedural memory = **knowing how**.

> **Key Study: HM (Scoville & Milner, 1957; Corkin, 1984)** - Henry Molaison (surgery 23 August 1953) had bilateral removal of the medial temporal lobes including the hippocampus. He developed **severe anterograde amnesia** (could not form new declarative memories: did not recognise staff, did not know where he lived). Yet he retained the ability to learn motor skills: mirror-tracing, pursuit rotor, and perceptual identification (non-declarative memory). HM's case: (1) established multiple LTM systems; (2) showed STM and LTM are dissociable; (3) demonstrated the role of the hippocampus and medial temporal lobes in declarative memory formation.

### Amnesic Syndrome (Features)
- **Anterograde amnesia**: severely impaired ability to form new memories after amnesia onset.
- **Retrograde amnesia**: impaired recall of pre-onset events (typically with a temporal gradient - older memories more intact).
- Short-term memory (digit span) largely **intact** (Spiers et al., 2001 - reviewed 147 cases).
- Residual skill learning ability (procedural, priming) largely **intact**.

### Episodic vs. Semantic Memory (Tulving, 1972)

| Feature | Episodic Memory | Semantic Memory |
|---------|----------------|-----------------|
| Content | Personal experiences at a specific time and place ("mental time travel") | General world knowledge; concepts; language; facts abstracted from experience |
| Time stamp | Yes - when and where | No - timeless |
| Key brain region | **Hippocampus** (and medial temporal lobes) | Underlying entorhinal, perirhinal, parahippocampal cortices; temporal lobe |
| Amnesia effect | Always impaired in hippocampal amnesia | Often relatively preserved |
| Retrograde gradient | Steep (recent > remote) | Shallow (only recent knowledge impaired) |

- Vargha-Khadem et al. (1997): patients Beth and Jon had bilateral hippocampal damage from birth/age 4. Both had **grossly deficient episodic memory** but attended ordinary schools with near-normal **semantic memory** - supporting episodic-semantic dissociation.
- Patient KC (Tulving, 2002): lost all episodic memories; semantic knowledge acquired before accident largely intact.
- Spiers et al. (2001) - 147 amnesic cases: episodic memory impaired in ALL; many had less severe semantic memory deficits.

### Episodic Memory and Future Thinking
- Same brain network (hippocampus, default mode network) supports both **recalling past events** and **imagining future events**.
- Hassabis et al. (2007): amnesic patients with hippocampal damage showed fragmentary, spatially incoherent imaginary future events.
- Viard et al. (2012) meta-analysis: hippocampal activation is typically greater for future imagination than for past recall.

### Semantic Memory: Concept Organisation
- Rosch et al. (1976): three-level hierarchy - **superordinate** (furniture) → **basic level** (chair) → **subordinate** (easy chair). Basic-level categories are preferred in everyday life: most informative while remaining distinctive.
- Barsalou's **situated simulation theory**: concept processing involves the perceptual and motor systems; representation varies with current goals and context. Hauk et al. (2004): words like "lick," "pick," "kick" activate overlapping areas of the motor strip (tongue, finger, foot).

### Non-Declarative Memory

**Priming** (repetition priming):
- **Perceptual priming**: facilitated processing of physical features of a repeated stimulus. Intact in most amnesiacs (Cermak et al., 1985; EP patient - Insausti et al., 2013).
- **Conceptual priming**: facilitated processing of meaning. Levy et al. (2004): amnesic patients EP and GP showed normal conceptual priming despite severely impaired recognition memory.
- Mechanism: **repetition suppression** - reduced brain activity with repeated stimuli, linked to more efficient neural processing (Gotts et al., 2012 synchrony model).
- Brain regions: early occipital areas (perceptual priming); left inferior frontal gyrus / left prefrontal cortex (conceptual priming).

**Procedural memory (skill learning)**:
- Spiers et al. (2001) - 147 amnesic cases: NONE was impaired on skills, habits, priming, or classical conditioning.
- Tranel et al. (1994): 28 amnesic patients all showed normal pursuit rotor learning; even patient Boswell (extensive LTM brain damage) showed two-year retention comparable to controls.
- Anderson et al. (2007): two severely amnesic patients showed normal car-driving skills.
- Key distinction: priming is **stimulus-specific** and **rapid**; skill learning **generalises** across stimuli and is **slow/gradual** (Knowlton & Foerde, 2008).

**Interacting systems**: The hippocampus is NOT exclusively for declarative memory. Foerde et al. (2006): weather prediction task - single-task condition correlated with medial temporal lobe activity (declarative); dual-task condition correlated with striatum (non-declarative). Stressed participants rely more on non-declarative (striatal) memory (Schwabe & Wolf, 2012).

## 3.8 Everyday Memory

### Autobiographical Memory
**Autobiographical memory**: long-term memory for the events of one's own life. Distinguished from episodic memory by: personal significance, longer time span (years to decades), integration of episodic and semantic components.

Functions (Bluck & Alea, 2009): (1) maintenance of social bonds; (2) directing future behaviour; (3) creating a sense of self-continuity.

### Memories Across the Lifespan (Rubin et al., 1986)

| Phenomenon | Description |
|-----------|-------------|
| **Childhood (infantile) amnesia** | Almost total lack of autobiographical memories from before age 3; limited recall ages 3-6 |
| **Reminiscence bump** | Disproportionately many memories from ages 10-30 (especially 15-25) in adults recalling their past |

- Childhood amnesia explanations: Freud's repression (rejected; fails to explain loss of positive/neutral memories); **cognitive self** hypothesis (Howe & Courage, 1997): autobiographical memory emerges after visual self-recognition (~age 2); **social-cultural developmental theory** (Fivush & Nelson, 2004): language and elaborative mother-child reminiscing scaffold autobiographical memory development.

### Flashbulb Memories
- **Brown and Kulik (1977)**: vivid, detailed memories for dramatic events (e.g., 9/11); proposed a special "Now Print!" neural mechanism.
- Six typical components: informant, place, ongoing event, own emotional state, others' emotional state, consequences.
- Evidence against special status: Talarico and Rubin (2003) - flashbulb memories showed the same consistency over 32 weeks as ordinary memories, despite being rated as more vivid.
- Evidence for partial specialness: Kvavilashvili et al. (2009) - 9/11 memories reasonably consistent for 3 years; Sharot et al. (2007) - those closer to the World Trade Center (more intense emotional experience) had more vivid, detailed memories and greater amygdala activation.
- **Rimmele et al. (2012)**: high consistency for location (83%), lower for immediate reaction (34%); negative emotion selectively enhances memory for SOME features, creating an illusion of complete remembering.

## 3.9 Eyewitness Testimony

### Accuracy of Eyewitness Memory
- In the US, 200+ wrongly convicted individuals have been exonerated by DNA evidence, mostly due to mistaken eyewitness identification.
- Simons and Chabris (2011): 37% of Americans believe a single confident eyewitness should suffice for conviction.
- Jurors disagree with eyewitness experts on 87% of issues; judges on 60% (Benton et al., 2006).

> **Key Study: Loftus and Palmer (1974)** - Eyewitnesses watched a film of a car accident and were asked about speed using "smashed" vs. "hit." "Smashed" → 41 mph; "hit" → 34 mph. One week later, 32% of the "smashed" group (vs. 12% of controls) falsely reported broken glass. Demonstrates the **misinformation effect**: post-event language alters memory for an event.

### Misinformation Effect
- **Misinformation effect**: post-event misleading information distorts eyewitness memory.
- Present even when eyewitnesses are explicitly warned (Ecker et al., 2010), though warnings reduce its size.
- **Mechanisms**:
  1. **Source misattribution**: eyewitness confuses where information came from (Lindsay et al., 2004 - similar-source events produce more intrusions).
  2. **Reconsolidation**: reactivating memory makes it fragile; misinformation overwrites the original trace (Chan & LaPaglia, 2013; Edelson et al., 2011 - amygdala-hippocampus connectivity predicts genuine vs. compliance-based memory change).
  3. **Vacant slot**: misinformation fills gaps in the original encoding.
  4. **Blend**: original and post-event information merge.

### Weapon Focus and Stress
- **Weapon focus** (Loftus et al., 1987): eyewitnesses fixate on the weapon, reducing memory for peripheral details (e.g., the culprit's face).
- Fawcett et al. (2013) meta-analysis: moderate weapon focus effect; similar size in lab and real-world studies.
- **Deffenbacher et al. (2004)** meta-analysis: culprit identification 54% in low-stress vs. 42% in high-stress conditions; details recalled 64% vs. 52%.
- Easterbrook's (1959) hypothesis: stress narrows attention to central stimuli, impairing peripheral detail memory.

### Schema Effects
- Bartlett (1932): schemas distort memory toward "what must have been true." Tuckey and Brewer (2003): eyewitnesses' recall of a bank robbery was systematically skewed toward bank-robbery schema content; ambiguous details (e.g., gender of a masked robber) were recalled as schema-consistent.

### Confirmation Bias
- Lindholm and Christianson (1998): Swedish participants were twice as likely to select an innocent immigrant as a culprit, reflecting expectations from crime statistics.

### Enhancing Eyewitness Memory: Cognitive Interview

**Geiselman and Fisher (1997)** - four retrieval rules:
1. **Mental reinstatement** of environment and personal context during the crime.
2. **Report everything** including minor details.
3. **Describe in multiple temporal orders** (e.g., backwards).
4. **Change perspective** (recall from different viewpoints).

- Theoretical basis: rules 1-2 based on **encoding specificity principle** (Tulving, 1979); rules 3-4 based on the multiplicity of retrieval routes in complex memory traces.
- **Enhanced cognitive interview** (Fisher & Geiselman, 1992): adds rapport building, slowing speech, reducing anxiety.
- Memon et al. (2010) meta-analysis: cognitive interview substantially increases **correct** recall; small increase in incorrect details; reduced effectiveness under high arousal or long retention intervals.
- Sequential > simultaneous line-ups: reduce false identifications (54% → 32% in culprit-absent conditions) with only an 8% reduction in correct identifications (Steblay et al., 2011).

## 3.10 Prospective Memory

**Prospective memory**: remembering to carry out an intended action at the appropriate future time or occasion, without an explicit reminder. Contrast with **retrospective memory** (memory for the past).

### Types

| Type | Definition | Performance |
|------|-----------|-------------|
| **Event-based** | Act when a specific external event occurs (e.g., "give this book to John when I see him") | Generally better (~52%) |
| **Time-based** | Act at a specific time (e.g., "phone Mary at 8 pm") | Generally worse (~33%) |

- Sellen et al. (1997): event-based PM = 52%; time-based PM = 33% in a naturalistic work environment.
- Event-based tasks are easier because external cues trigger retrieval; time-based tasks require sporadic self-initiated monitoring that intensifies as the deadline approaches.

### Stages of Prospective Memory (Zogg et al., 2012)
1. **Intention formation**: encode the intention linked to a cue.
2. **Retention interval**: environmental monitoring for task-relevant cues (minutes to weeks).
3. **Cue detection and intention retrieval**: recognise the cue; self-initiate retrieval.
4. **Intention recall**: retrieve the intention from retrospective memory.
5. **Intention execution**: typically automatic.

- PM failures are interpreted as reflecting poor reliability/motivation (not just poor memory); distinct from retrospective memory failures (Graf, 2012).

## Key Terms

| Term | Definition |
|------|-----------|
| **Iconic memory** | Sensory store for visual information; decays within ~500 ms |
| **Echoic memory** | Sensory store for auditory information; persists ~2 s |
| **Chunk** | Stored unit formed from integrating smaller pieces of information |
| **Multi-store model** | Atkinson and Shiffrin (1968): sensory stores, STS, LTS |
| **Primacy effect** | Better recall of early list items; transferred to LTM via rehearsal |
| **Recency effect** | Better recall of final list items; retained in STS |

## Likely Exam Points

1. **Atkinson-Shiffrin model**: three stores (sensory, STS, LTS). Capacity: sensory (large, modality-specific), STS (~7 items but 3-4 true chunks), LTS (unlimited). Duration: iconic ~500 ms, echoic ~2 s, STS seconds, LTS years.

2. **Double dissociation supporting STS/LTS distinction**: KF (intact LTM, severely impaired STS) vs. HM/amnesics (intact STS/digit span, severely impaired LTM). Both directions of dissociation confirm the stores are separate.

3. **Working memory model - four components**: phonological loop (phonological store + articulatory rehearsal), visuo-spatial sketchpad (visual cache + inner scribe), central executive (inhibition/shifting/updating; prefrontal cortex), episodic buffer (added 2000; 4 chunks; passive; integrates information).

4. **Phonological similarity effect**: recall worse for phonologically similar words - items stored in phonological code. **Word-length effect**: shorter words recalled better - longer words exhaust rehearsal loop. **Articulatory suppression** (saying "the the the") eliminates the word-length effect for visual input.

5. **Baddeley et al. (1998) - phonological loop and foreign vocabulary**: Patient PV (digit span = 2) could not learn Russian-Italian word pairs but could learn Italian-Italian pairs. Function of the phonological loop = learning new word forms (foreign vocabulary).

6. **Miyake et al. (2000) - three executive functions**: inhibition, shifting, updating. **Dysexecutive syndrome** = frontal damage causing all three to be impaired. Unity/diversity framework: inhibition has no unique variance over the common executive factor.

7. **Levels of processing (Craik & Lockhart, 1972)**: shallow (structural/graphemic) < phonemic < deep (semantic). Craik and Tulving (1975): 3x better recognition with deep vs. shallow incidental learning. **Elaboration** and **distinctiveness** also improve retention independently of depth.

8. **Transfer-appropriate processing (Morris et al., 1977)**: on a rhyming recognition test, shallow encoding beats deep encoding - the best encoding matches the demands of the test.

9. **Testing effect (Roediger & Karpicke, 2006)**: repeated testing > repeated studying for 1-week retention (50% advantage). Mediator effectiveness hypothesis: retrieval practice improves quality of encoded mediators.

10. **Ebbinghaus forgetting curve**: rapid early forgetting, then deceleration. Measured with the savings method (fewer relearning trials = better retention). Implicit memory shows slower forgetting than explicit.

11. **Proactive interference (PI)**: old learning disrupts new. **Retroactive interference (RI)**: new learning disrupts old. Both maximal when different responses paired with the same stimulus. Intentional forgetting (directed forgetting, think/no-think) reduces PI by restricting retrieval search.

12. **Encoding specificity principle (Tulving, 1979)**: memory depends on overlap between encoding and retrieval context. Evidence: Godden and Baddeley (1975) - divers recalled better in the same environment. Prediction: recall can sometimes exceed recognition when recall cues better match the trace.

13. **Consolidation theory**: new memories are fragile; hippocampus consolidates → neocortex stores. Predicts temporal gradient in retrograde amnesia (recent memories more impaired). **Reconsolidation**: reactivating a memory makes it fragile again and updatable.

14. **HM case**: bilateral medial temporal lobe/hippocampus removal → severe anterograde declarative amnesia; intact non-declarative (mirror tracing, pursuit rotor, perceptual identification). Demonstrates multiple LTM systems and hippocampal role in declarative memory.

15. **Declarative vs. non-declarative memory**: declarative (explicit; episodic + semantic; conscious recollection; hippocampus/MTL). Non-declarative (implicit; procedural + priming; no conscious recollection; striatum for skills; occipital/frontal for priming). Amnesics: impaired declarative, intact (or near-intact) non-declarative (Spiers et al., 2001 - 147 cases; Tranel et al., 1994 - pursuit rotor).

16. **Episodic vs. semantic memory**: episodic = time-stamped, personal events, hippocampus-dependent. Semantic = timeless general knowledge, relatively preserved in hippocampal amnesia. Vargha-Khadem et al. (1997): patients Beth and Jon - grossly impaired episodic, near-normal semantic. KC: no episodic memory, intact pre-accident semantic knowledge.

17. **Autobiographical memory**: childhood amnesia (before age 3); reminiscence bump (ages 10-30). Flashbulb memories: subjectively vivid but not necessarily more accurate or consistent than ordinary memories (Talarico & Rubin, 2003). Intensity of emotional experience matters (Sharot et al., 2007: those closer to World Trade Center had more vivid memories and greater amygdala activation).

18. **Misinformation effect (Loftus & Palmer, 1974)**: post-event language ("smashed" vs. "hit") alters speed estimates and false memory for broken glass. Mechanisms: source misattribution and reconsolidation. Weapon focus: reduces memory for culprit and peripheral details; confirmed in lab and real-world studies (Fawcett et al., 2013 meta-analysis). Stress reduces eyewitness accuracy (Deffenbacher et al., 2004: 12% reduction in identification).

19. **Cognitive interview (Geiselman & Fisher, 1997)**: four rules - mental reinstatement, report everything, multiple temporal orders, change perspective. Based on encoding specificity (rules 1-2) and multiple retrieval routes (rules 3-4). Meta-analysis (Memon et al., 2010): substantially more correct detail; small increase in incorrect detail. Sequential line-ups superior to simultaneous (reduce false ID rate by 22%, with only 8% loss of correct IDs).

20. **Prospective memory**: event-based (triggered by an external event; ~52% correct) > time-based (triggered by a time; ~33% correct). Five stages: intention formation → retention interval → cue detection/intention retrieval → intention recall → intention execution. Failures attributed to poor motivation/reliability (not poor memory).
